## P9 衔接对照表示范（供进阶班参考格式）

进阶班 P9 产出时，照以下格式输出，每条标明本课程证据 + 对应扎根班概念：

```json
[
  {
    "本课程概念": "（进阶班填写）",
    "衔接类型": "延续 / 深化 / 新引入",
    "前序对应概念": "（对应扎根班概念名或别名）",
    "判断依据": "（如：扎根班已引入但停在觉知/接纳层面；进阶班往发愿/行愿推进，属深化）",
    "本课程证据": ["进阶班slice_id"],
    "前序证据": "扎根班概念卡路径或slice_id"
  }
]
```

## 使用说明（进阶班 P9 操作流程）

1. 进阶班 P5 完成后，拿到进阶班概念清单。
2. 读取本文件（扎根班概念清单）。
3. 按概念名 + 别名做初步匹配，先找"可能对应同一概念"的候选对。
4. 对每个候选对，机器并列摆出两边的代表原话，不下衔接判断。
5. 由人逐条判断：延续（讲法一致）/ 深化（更进一步）/ 新引入（扎根班没有）。
6. 产出进阶班版的 P9 衔接对照表，补充进进阶班的体系产物。
